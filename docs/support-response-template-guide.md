# Support Response Template Guide

`scripts/generate_support_response_template.py` turns the response rubric and session note into a ready-to-fill support reply template.

Use it when you want a consistent structure for GitHub issues, forum responses, or partner follow-ups.

```bash
make support-response-template
```

Outputs:

- `outputs/support_response_template/support_response_template.json`
- `outputs/support_response_template/support_response_template.md`
